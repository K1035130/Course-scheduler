# Course-scheduler
- Hackathon project
- A lightweight web application designed to assist UBC students with course selection. It lets students enter course meeting times and instantly builds a conflict-free timetable.

## How to run

Open https://course-scheduler-0omv.onrender.com in your browser.
The webside is only a demo for now, It contains only a limited amount of course data for the second semester of the 2026 winter term, and the AI-suggestion API will cease to function in May 2026.
